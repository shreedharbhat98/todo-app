## About
It is a simple todo application for daily uses where the user can manage his tasks.


## Getting started
### To run that demo on your own computer:
* `Clone this repository`
* `npm install`

Install dependencies :

* `@material-ui/core`
* `@material-ui/icons`
* `@material-ui/lab`
* `@material-ui/system`
* `react-redux`
* `redux`
* `uuid`

Run command 
* `npm start`
